## [1.0.0] — Coming Soon

### Added
- Interactive CLI with arrow-key navigation
- Three framework levels — Beginner, Intermediate, Advanced
- Auto-generated `pom.xml` with correct dependency versions
- `DriverFactory.java` with Chrome, Firefox, Edge support
- `BaseTest.java` with `@BeforeMethod` / `@AfterMethod` setup
- Auto-install for Java 21 (Temurin), Maven 3.9.9, Git
- Auto-update on every run — pulls latest from GitHub
- Offline fallback — runs on last downloaded version
- Full test data setup — Excel, JSON, CSV, Properties
- ExtentReports, Allure, Log4j2, REST Assured support (Intermediate+)
- ThreadLocal `DriverManager` for parallel execution (Advanced)
- Page Object Model with `BasePage` and `AppPage` (Advanced)
- Cross-browser TestNG suite files auto-generated
- Git config setup from terminal — no IDE needed
- Supports Linux, macOS, Windows (Git Bash)
